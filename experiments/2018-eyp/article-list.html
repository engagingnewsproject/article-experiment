Logs

For each new user, we generate a userID and store it in localStorage (like sessionStorage but persists across windows/tabs). This gets passed to the log for every interaction so we can see what a user does across their entire visit.

The breakdown of each log is like this:

[ url of the page they're on ]|[ id of the page ]|[ user ID ]|[ user IP ]|[ date logged ]|[ action category ]|[ action ]|[ description ]

For example, here's the log when a page is loaded:

https://eyp-study.test/articles/hit-and-run/ctr.php|hr-ctr|db_5b2bd7ed3b158|127.0.0.1|07/31/2018 8:18:04 AM|Page View|Police-search-for-suspect-in-near-fatal-hit-and-run|Loaded

Here are all the [ action category ]|[ action ]|[ description ]'s we are logging:

  • When a page gets loaded:
    Page View|[ url of the loaded page ]|Loaded
  • When a Twitter for Facebook share button is clicked:
    click|Button: [ button type ]|[ button location ]
  • When a user fills out and successfully submits the comment form:
    Comment|Add Comment|[ name from comment form ]: [ comment text ]
